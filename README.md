# ♟️ C++ Console Chess Game
### Programming Fundamentals Final Project

This is a fully functional, text-based Chess game developed in C++ as my final project for the Programming Fundamentals course. It focuses on implementing complex game logic, file handling, and basic AI decision-making.

## 🚀 Features
* **AI Opponent:** Includes a built-in AI mode so you can challenge the computer.
* **Full Piece Logic:** movement rules for Pawns, Rooks, Knights, Bishops, Queens, and the King.
* **Turn-Based Play:** A two-player experience managed through the console.
* **Move Validation:** Ensures players can only make legal moves according to standard chess rules.
* **Save & Load System:** Uses external `.txt` files (`save`) to store and resume game states.
* **Undo/Redo Functionality:** Uses the `undo_redo` file to allow players to navigate through their move history.

## 🛠️ How to Run (Visual Studio)
To play this game on your local machine:
1. Download or copy the code from the `chessgame.cpp` file in this repository.
2. **Important:** Download the `save` and `undo_redo` text files as well.
3. Open **Visual Studio** and create a new **C++ Console App** project.
4. Paste the code into your `source.cpp` file.
5. Place the `save` and `undo_redo` files in the **same folder** as your source code.
6. Press **F5** to build and run the game!
