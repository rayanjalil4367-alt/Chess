# ♟️ C++ Console Chess Game
### Programming Fundamentals Final Project

This is a fully functional, text-based Chess game developed in C++ as my final project for the Programming Fundamentals course. It focuses on implementing complex game logic through procedural programming.

## 🚀 Features
* **Full Piece Logic:** Includes movement rules for Pawns, Rooks, Knights, Bishops, Queens, and the King.
* **Turn-Based Play:** A two-player experience managed through the console.
* **Move Validation:** Ensures players can only make legal moves according to standard chess rules.
* **Logic-Driven:** Built using 2D arrays, loops, and conditional structures.

## 🛠️ How to Run (Visual Studio)
To play this game on your local machine:
1. Copy the code from the `chess_game.cpp` file in this repository.
2. Open **Visual Studio**.
3. Create a new **C++ Console App** project.
4. Paste the code into your `source.cpp` file.
5. Press **F5** to build and run the game!
